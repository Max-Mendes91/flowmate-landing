# FlowMate

**FlowMate** is an AI assistant available as a desktop SaaS app and inside messaging apps like Telegram and WhatsApp. It helps freelancers and consultants manage email, scheduling, and client communication without switching apps.

One AI assistant for your inbox, calendar, and client communication ‚Äî available on desktop, Telegram, and WhatsApp.
